# Simple To-Do List Application

A lightweight, responsive todo list i built for fun.

## 🚀 Features

*   **Add Tasks:** Enter task descriptions dynamically via the main input bar.
*   **Complete Tasks:** Toggle a visual strike-through effect by clicking on items.
*   **Delete Tasks:** Remove individual entries permanently with a single click.

## 🛠️ Tech Stack

*   **HTML5:** Structure and base markup.
*   **CSS3:** Custom interface layout and responsive styling.
*   **JavaScript (ES6+):** Interactive logic and state updates.

## 📂 Project Structure

```
├── index.html       # Application interface framework
```

### Running the Application

1. Clone this repository or download the source code files.
2. Navigate into your project folder root.
3. Double-click the `index.html` file to launch the app instantly in your browser.
